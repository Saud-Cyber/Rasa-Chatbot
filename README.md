Chatbot Project
A simple configurable chatbot using structured files and a web interface. Easily customizable for demonstrations, learning, and quick deployment in various environments.

Features
Modular setup with YAML configuration files for intents, domains, and endpoints.

Intuitive web chat interface (index.html) for testing and integration.

Simple to extend, making it ideal for beginners and prototyping.

Project Structure
index.html — Main web interface for the chatbot.

config.yml — Chatbot configuration parameters.

domain.yml — Defines intents, actions, and responses.

endpoints.yml — Configuration for API or model endpoints.

credentials.yml — Stores authentication info for external services.
